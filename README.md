# Calculate_OLED
## ESP32 + OLED的计算器函数绘图项目


1. 计算器函数编写
2. 二元一次函数绘制
3. 一元二次解方程
4. 按键定位

## 使用方法
1. 安装VScode
2. 安装插件platformio
3. 导入libraries (Adafruit BusIO, Adafruit GFX Library, Adafruit SSD1306)

接线
```
#define OLED_MOSI 16
#define OLED_CLK 13
#define OLED_DC 4
#define OLED_CS 23
#define OLED_RESET 0
```
